Consolidation

How Kortny turns recent activity into lasting memory. Each run distills the day's tasks and observations into durable facts, merges duplicates, and files away what changed.

{{ page_intro("Consolidation is Kortny's sleep cycle. Raw experience — task episodes, channel observations — piles up during the day; when enough is new and the workspace goes quiet, a run distills it into durable knowledge. Each run executes independent passes: promotion (an LLM arbitrates every episode against existing knowledge — add, update, invalidate, or ignore; most things are ignored, chitchat doesn't become memory), candidate adjudication, duplicate merge, aging, fact reconciliation, retention hygiene, per-channel style cards, and embedding backfill. Contradictions never delete: the old belief is timestamped invalid and a successor is created. This page shows every run with per-pass counters and LLM cost — if memory changed, you can see exactly when and why.") }}
